# 01 PL/SQL 기본 구조
## 블록
- PL/SQL 소스 프로그램의 기본 단위
- 선언부, 실행부. 예외 처리부로 구성
- 익명 블록 - 함수, 프로시저. 패키지
```sql
이름부
IS(AS)
    선언부
BEGIN
    실행부
EXCEPTION
    예외 처리부
END;
```

1. 이름부: 생략하면 익명 블록이 된다
2. 선언부 : DECALRE로 시작되며 실행부와 예외 처리부에서 사용할 각종 변수, 상서, 커서 등을 의미
3. 실행부 : 실제 로직을 처리하는 부분
4. 예외 처리부 : 로직을 처리하다가 오류가 발생하면 처리할 내용을 기술하는 부분
